Estruturas de Repetição em JavaScript

Estruturas de repetição são essenciais para a automação de tarefas repetitivas dentro de um programa. Elas permitem que você execute um bloco de código várias vezes, sem precisar escrever o mesmo código repetidamente. JavaScript oferece três tipos principais de estruturas de repetição: for, while e do...while. Vamos entender como cada uma funciona e quando usá-las.

1. Estrutura for

A estrutura for é usada quando o número de iterações é conhecido antecipadamente. Ela é especialmente útil quando precisamos iterar sobre arrays ou executar um bloco de código um número fixo de vezes.

Sintaxe:

for (inicialização; condição; incremento) {
    // bloco de código a ser executado
}
  • inicialização: Executada apenas uma vez, antes da primeira iteração. Geralmente usada para inicializar variáveis.
  • condição: A cada iteração, é verificada. Se for verdadeira, o bloco de código é executado. Se for falsa, o laço é interrompido.
  • incremento: Executado após cada iteração, é utilizado para modificar a variável de controle.

Exemplo de uso do for:


for (let i = 0; i < 5; i++) {
    console.log("Iteração número " + i);
}

O código acima imprime as mensagens "Iteração número 0" até "Iteração número 4" no console.

2. Estrutura while

A estrutura while é usada quando não sabemos o número exato de iterações, mas sabemos que a repetição deve continuar enquanto uma condição for verdadeira. A condição é verificada antes de cada iteração.

Sintaxe:

while (condição) {
    // bloco de código a ser executado
}
  • condição: A condição é verificada antes de cada iteração. Se for verdadeira, o código dentro do bloco será executado. Caso contrário, o laço é interrompido.

Exemplo de uso do while:


let i = 0;
while (i < 5) {
    console.log("Iteração número " + i);
    i++;
}

O código acima imprime as mensagens "Iteração número 0" até "Iteração número 4", de forma semelhante ao exemplo do for. A diferença é que aqui a variável i é incrementada manualmente dentro do bloco de código.

3. Estrutura do...while

A estrutura do...while é semelhante à estrutura while, mas a principal diferença é que o bloco de código será executado pelo menos uma vez, mesmo que a condição seja falsa. A condição é verificada após a execução do bloco.

Sintaxe:

do {
    // bloco de código a ser executado
} while (condição);
  • condição: A condição é verificada após a execução do bloco de código. Se for verdadeira, o bloco será executado novamente. Se for falsa, o laço será interrompido.

Exemplo de uso do do...while:


let i = 0;
do {
    console.log("Iteração número " + i);
    i++;
} while (i < 5);

O código acima também imprime as mensagens "Iteração número 0" até "Iteração número 4", mas, como a condição é verificada após a execução do código, o laço será executado pelo menos uma vez, independentemente da condição.

Exemplos Práticos

A seguir, você pode testar os exemplos de código no Visual Studio Code para ver como as estruturas de repetição funcionam na prática.

Exemplo prático 1: Exibir os números de 1 a 10 usando o for.


for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Exemplo prático 2: Imprimir uma mensagem até que o número seja maior que 5 usando o while.


let i = 1;
while (i <= 5) {
    console.log("Número é " + i);
    i++;
}

Exemplo prático 3: Fazer o código rodar pelo menos uma vez com do...while.


let i = 0;
do {
    console.log("Número é " + i);
    i++;
} while (i < 0);  // A condição é falsa, mas o loop ainda ocorre uma vez.

Exercícios

Agora que você aprendeu sobre as estruturas de repetição em JavaScript, pratique com os seguintes exercícios:

  1. Utilize o for para imprimir todos os números de 1 a 100 que são divisíveis por 3.
  2. Crie um programa com while que leia um número até que o usuário digite um número negativo.
  3. Implemente um do...while que exibe "Continuando..." até que o número de tentativas seja 3.

Conclusão

Estruturas de repetição são fundamentais para controlar o fluxo de execução em programas de forma eficiente. Cada tipo de laço tem uma utilidade específica, dependendo da situação em que você se encontra:

  • for: quando você sabe quantas vezes precisa repetir algo.
  • while: quando a repetição depende de uma condição que pode variar durante a execução.
  • do...while: quando você quer garantir que o código seja executado pelo menos uma vez.
